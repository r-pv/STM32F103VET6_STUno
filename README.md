# STM32F103VET6 STUno

An ST development board based on the STM32F103VET6/STM32F103VE (Clock: 72MHz, flash: 512Kb, Ram:64Kb) MCU (the LQFP-100 version of the famous Blue Pill) compatible with STM32Duino (Arduino).
